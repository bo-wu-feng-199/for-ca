介绍
🧮
现金结算辅助工具

输入商品价格和支付金额，自动计算支付差额并给出最优货币组合方案，无需手动心算。

⚡
实时计算

输入即算，300ms 防抖，三种方案（最优 / 均衡 / 实操）同步生成。

📦
轻量离线

纯前端实现，无后端依赖。数据存储在浏览器本地，无需网络即可使用。

📋
记录与导出

自动保存最近 10 条计算记录，支持 JSON / CSV / Markdown 三种格式导出。

使用
1
输入价格与支付金额

选择双框模式分别输入，或切换到单行模式用空格 / + 号 / 逗号分隔输入。

2
选择分类标签

为本次计算选择一个标签（购物 / 餐饮 / 出行 / 收银 / 其他），方便后续筛选。

3
查看三种方案

最优方案张数最少、均衡方案分布均匀、实操方案减少小币。点击历史记录可一键复用。

4
导出与整理

在"计算记录"区域点击 JSON / CSV / MD 按钮导出数据，用于归档或分析。

注意事项
  • 金额上限为 ¥1,000，超出范围将提示错误。
  • 计算基于人民币（¥）面额体系：100 / 50 / 20 / 10 / 5 / 1 / 0.5 / 0.1 / 0.05 / 0.01。
  • 所有数据存储在浏览器 localStorage 中，清除浏览器缓存会丢失记录。
  • 导出功能基于浏览器 Blob API，不会将数据上传到任何服务器。
  • Dark Mode 切换不影响数据，仅改变界面外观。
  • 本工具仅为辅助计算，实际交易请以双方确认为准。
